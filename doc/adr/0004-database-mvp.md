# ADR-0004: База данных (MVP) — SQLite, SQLAlchemy 2

## Статус
Принято

## Контекст
Нужна БД для MVP без избыточной инфраструктуры; возможность позже масштабировать.

## Решение
- **БД (MVP)**: SQLite
- **ORM**: SQLAlchemy 2
- **Миграции**: Alembic
- **После MVP**: возможен переезд на Postgres без смены моделей и слоёв приложения.

## Последствия
- Один файл БД, не нужен отдельный сервер БД на старте.
- Модели и миграции готовы к переносу на Postgres при росте нагрузки.
