# ADR-0001: Формат продукта — Telegram Mini App + Bot

## Статус
Принято

## Контекст
Нужно выбрать формат представления продукта пользователям Telegram: только бот, только веб-приложение или комбинация.

## Решение
Используем **Telegram Mini App (WebApp)** и **Bot** вместе:
- **Bot**: входная точка, меню, deep-links, уведомления.
- **Mini App**: основной UI (рынок, портфель, депозит, сделки).

## Последствия
- Пользователь открывает приложение через бота; бот даёт кнопку/ссылку на Mini App.
- Весь интерактивный интерфейс — в Mini App; бот используется для входа и уведомлений.
- Нужен HTTPS для Mini App и webhook бота.
