# ADR-0002: Backend — Python 3.12, FastAPI

## Статус
Принято

## Контекст
Нужен стек для API: язык, фреймворк, минимальная сложность для MVP.

## Решение
- **Язык**: Python 3.12
- **API**: FastAPI

## Последствия
- Быстрая разработка, встроенная валидация и документация OpenAPI.
- Один процесс (API) обслуживает запросы Mini App и webhook'и (в т.ч. TON).
