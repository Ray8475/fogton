# ADR-0005: Frontend Mini App (MVP) — HTML, Tailwind, vanilla JS

## Статус
Принято

## Контекст
Нужен минимально сложный фронт для Mini App, чтобы быстрее проверить идею.

## Решение
- **UI**: HTML + Tailwind (CDN)
- **Логика**: vanilla JS
- Без React/Vue на MVP.

## Последствия
- Нет сборки и зависимостей фронта; быстрые итерации.
- При росте сложности UI можно позже ввести фреймворк.
