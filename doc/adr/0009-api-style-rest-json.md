# ADR-0009: Стиль API — REST + JSON

## Статус
Принято

## Контекст
Mini App нужен простой способ получать данные и отправлять действия в backend.

## Решение
- **REST + JSON**: небольшой набор простых эндпоинтов.
- Без GraphQL и RPC-слоёв на MVP.

## Последствия
- Простая интеграция, стандартные HTTP-методы и коды ответов.
- При росте сложности запросов можно рассмотреть GraphQL позже.
