# ADR-0003: Бот — aiogram

## Статус
Принято

## Контекст
Нужна библиотека для Telegram-бота: асинхронность, поддержка webhook, простота.

## Решение
Используем **aiogram** для реализации бота.

## Последствия
- Асинхронный бот, совместимый с async FastAPI при необходимости.
- Поддержка webhook и polling; команды и кнопки для открытия Mini App.
