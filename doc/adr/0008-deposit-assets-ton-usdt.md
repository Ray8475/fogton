# ADR-0008: Активы депозитов — TON и USDT (jetton) в сети TON

## Статус
Принято

## Контекст
Нужно выбрать сеть и валюты для пополнения баланса в приложении.

## Решение
- **Сеть**: TON
- **Активы**: TON (нативная монета) и USDT (jetton) в сети TON

## Последствия
- Депозиты обрабатываются через провайдера TON API и webhook в наш API.
- Внутренний учёт ведётся по валютам TON и USDT отдельно (balances, ledger_entries).
