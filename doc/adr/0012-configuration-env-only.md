# ADR-0012: Конфигурирование — только env-переменные

## Статус
Принято

## Контекст
Нужен простой способ настройки API, бота и окружения без лишних конфиг-файлов на MVP.

## Решение
- **Только env-переменные**: локально через `.env`, на деплое — переменные окружения.
- Конфиги по областям: Telegram (BOT_TOKEN, WEBHOOK_BASE_URL, …), DB (DATABASE_URL), TON, Security (JWT_SECRET, ADMIN_TOKEN).

## Последствия
- Один способ конфигурации для всех сред; секреты не в коде.
- Список переменных — в `.env.example`; при росте можно добавить валидацию (например Pydantic Settings).
